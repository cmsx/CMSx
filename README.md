CMSx
====================

[Официальный сайт](http://www.cmsx.ru).

Структура папок
---------------

* **common** - общие для системы ресурсы
* **files** - файлы сохраняемые в системе
* **html** - верстка
* **inc** - основная папка системы
  * **app** - файлы специфичные для приложения
  * **ctrl** - контроллеры
  * **lib** - библиотека базовых классов
  * **test** - тесты
  * **tmpl** - шаблоны
* **tmp** - временные файлы
